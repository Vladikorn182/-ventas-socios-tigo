AUTOGESTIÓN TIGO - ZIP INTEGRADO FINAL

Este paquete contiene una versión limpia en un solo proyecto Streamlit.

ARCHIVOS INCLUIDOS:
- app.py
- agenda_tecnica.py
- pendientes_inst.py
- suspendidas.py
- requirements.txt

MÓDULOS INCLUIDOS:
1) Agenda Técnica
   - Carga BO-CITA SERVICIO NACIONAL.
   - Filtra instalaciones por fecha y EH.
   - Genera mensaje global para WhatsApp del grupo.

2) Pendientes de Instalación con Pago
   - Carga PENDIENTE_INST_CON_PAGO.
   - Filtra casos antiguos de +3 días.
   - Muestra código, nodo, fecha, días, EH y socio.
   - Genera mensaje WhatsApp global.

3) Suspendidas
   - Carga archivo de suspendidas.
   - Filtra casos antiguos de +3 días.
   - Clasifica observación CRM.
   - Genera mensaje WhatsApp global.

IMPORTANTE PARA EVITAR EL ERROR DE STREAMLIT:
- No copies carpetas pages/ anteriores dentro de este proyecto.
- Si tu repositorio ya tiene pages/03_pendientes_inst.py o pages/04_suspendidas.py, elimínalas antes de subir este ZIP.
- Debe quedar un solo app.py y los tres módulos en la carpeta principal.

PASOS PARA SUBIR A GITHUB / STREAMLIT CLOUD:
1) Descomprime este ZIP.
2) Reemplaza en tu repositorio los archivos por estos cinco archivos.
3) Elimina cualquier carpeta pages/ vieja si causa duplicados.
4) Sube cambios a GitHub.
5) Reinicia la app en Streamlit Cloud.

Si quieres conservar dashboard, whatsapp, pendientes_pago u objetivos_configuracion, debes subir esos archivos reales para integrarlos también. Este ZIP solo incluye los módulos que enviaste y que se pudieron validar.
